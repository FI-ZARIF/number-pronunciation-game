# Number Pronunciation Game

## 📌 Project Description
The "Number Pronunciation Game" is a simple Java-based GUI application developed using Java Swing. The main purpose of this project is to convert numerical input (1–100) into its corresponding English word pronunciation. It provides an interactive and user-friendly interface for beginners to understand number-to-word conversion logic.

---

## 🎯 Objectives
- To build a simple GUI application using Java Swing  
- To accept numerical input from users (1–100)  
- To convert numbers into English word pronunciation  
- To display output in a user-friendly interface  
- To improve understanding of Java event handling and logic building  

---

## 🛠️ Technologies Used
- Programming Language: Java  
- Framework: Java Swing (GUI Framework)  
- IDE: Visual Studio Code / IntelliJ IDEA / Eclipse  
- JDK Version: Java SE (JDK 8 or above)
